回忆一下java基础数据类型，基础数据类型是java语言规范定义，与平台无关。

<!-- more -->

### 整型
| 类型  | 占用字节 | 占用bit | 数值范围                                                    |
| ----- | -------- | ------- | ----------------------------------------------------------- |
| byte  | 1        | 8       | -2^7 ~ 2^7-1 (-128 ~ 127)                                   |
| short | 2        | 16      | -2^15 ~ 2^15-1 (-32768 ~ 32767)                             |
| int   | 4        | 32      | -2^31 ~ 2^31-1 (-2147483648 ~ 2147483647)                   |
| long  | 8        | 64      | -2^63 ~ 2^63-1 (-9223372036854775808 ~ 9223372036854775807) |

借助java的整数包装类看下：

```java
public class IntegerDataType {

    public static void main(String[] args) {
        IntegerDataType integerDataType = new IntegerDataType();
        integerDataType.integer();
    }

    public void integer() {
        System.out.println("byte: " + Byte.BYTES + " bytes, " + Byte.SIZE + " bits");
        System.out.println("byte max = " + Byte.MAX_VALUE + " min = " + Byte.MIN_VALUE);
        System.out.println("short: " + Short.BYTES + " bytes, " + Short.SIZE + " bits");
        System.out.println("short max = " + Short.MAX_VALUE + " min = " + Short.MIN_VALUE);
        System.out.println("int: " + Integer.BYTES + " bytes, " + Integer.SIZE + " bits");
        System.out.println("int max = " + Integer.MAX_VALUE + " min = " + Integer.MIN_VALUE);
        System.out.println("long: " + Long.BYTES + " bytes, " + Long.SIZE + " bits");
        System.out.println("long max = " + Long.MAX_VALUE + " min = " + Long.MIN_VALUE);
    }
}
```

输出结果：

```
byte: 1 bytes, 8 bits
byte max = 127 min = -128
short: 2 bytes, 16 bits
short max = 32767 min = -32768
int: 4 bytes, 32 bits
int max = 2147483647 min = -2147483648
long: 8 bytes, 64 bits
long max = 9223372036854775807 min = -9223372036854775808
```

整型数据左边第1位为符号位，其他位为数据位，也是对应数值范围的最高幂次。

### 浮点型

| 类型   | 占用字节 | 占用bit |
| ------ | -------- | ------- |
| float  | 4        | 32      |
| double | 8        | 64      |

借助java的浮点数包装类看下：

```java
public class DecimalDataType {

    public static void main(String[] args) {
        DecimalDataType baseDataType = new DecimalDataType();
        baseDataType.decimal();
    }

    public void decimal() {
        System.out.println("float: " + Float.BYTES + " bytes, " + Float.SIZE + " bits");
        System.out.println("float positive max = " + Float.MAX_VALUE + " min = " + Float.MIN_VALUE + " min normal = " + Float.MIN_NORMAL);
        System.out.println("double: " + Double.BYTES + " bytes, " + Double.SIZE + " bits");
        System.out.println("double positive max = " + Double.MAX_VALUE + " min = " + Double.MIN_VALUE + " min normal = " + Double.MIN_NORMAL);
    }
}
```

输出结果：

```
float: 4 bytes, 32 bits
float positive max = 3.4028235E38 min = 1.4E-45 min normal = 1.17549435E-38
double: 8 bytes, 64 bits
double positive max = 1.7976931348623157E308 min = 4.9E-324 min normal = 2.2250738585072014E-308
```

浮点数存在最大正数MAX_VALUE、最小正正规值MIN_NORMAL，最小正非正规值MIN_VALUE，这是源于java中浮点数遵循IEEE 754标准（浮点数表示的国际标准）：

| 特性                            | 单精度（`float`）      | 双精度（`double`）       |
| ------------------------------- | ---------------------- | ------------------------ |
| **总位数（bit）**               | 32 位                  | 64 位                    |
| **符号位（sign）**              | 1 位                   | 1 位                     |
| **指数位（exponent）**          | 8 位                   | 11 位                    |
| **尾数位（fraction/mantissa）** | 23 位                  | 52 位                    |
| **有效数字精度**                | ≈ 7 位十进制数字       | ≈ 15~17 位十进制数字     |
| **数值范围（近似）**            | ±1.2×10⁻³⁸ ~ ±3.4×10³⁸ | ±2.2×10⁻³⁰⁸ ~ ±1.8×10³⁰⁸ |
| **Java 类型**                   | `float`                | `double`                 |

尾数（mantissa）实际有 **24 位/53 位精度**，因为 IEEE 754 使用“隐含前导 1”。

### 字符型

| 类型 | 占用字节 | 占用bit | 数值范围         |
| ---- | -------- | ------- | ---------------- |
| char | 2        | 16      | 0 ~ 2^16 (65535) |

借助java的字符包装类看下：

```java
public class CharacterDataType {

    public static void main(String[] args) {
        CharacterDataType dataType = new CharacterDataType();
        dataType.character();
    }

    public void character() {
        System.out.println("char: " + Character.BYTES + " bytes, " + Character.SIZE + " bits");
        System.out.println("char max = " + (int) Character.MAX_VALUE + " min = " + (int) Character.MIN_VALUE);

        // 十六进制形式
        System.out.println("char hex max = \\u" + Integer.toHexString(Character.MAX_VALUE).toUpperCase());
        System.out.println("char hex min = \\u" + Integer.toHexString(Character.MIN_VALUE).toUpperCase());
    }
}
```

输出结果：

```
char: 2 bytes, 16 bits
char max = 65535 min = 0
char hex max = \uFFFF
char hex min = \u0
```

char是一种基本数据类型，用于表示16位的Unicode字符，它的数值范围是固定且无符号的，可以表示 Basic Multilingual Plane (BMP) 中的所有字符。

### 布尔型

Java语言规范没有规定boolean的具体大小。JVM实现通常将单个boolean变量存储为1个字节（8bits），因为：
CPU内存寻址最小单位是字节（byte），无法直接访问1bit。用1byte存储true/false更高效（避免位操作开销）。
